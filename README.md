# aws-static-website
🌐 AWS Static Website Hosting Project

📌 Project Overview

This project demonstrates how to host a static website using AWS cloud services. The website is deployed using Amazon S3 and delivered globally with low latency using Amazon CloudFront.


## 🏗 Architecture
User → CloudFront → S3 Bucket



🚀 AWS Services Used

- Amazon S3 – Used for storing and hosting static website files
- Amazon CloudFront – Used as a Content Delivery Network (CDN) for fast content delivery





🛠️ Features

- Responsive static portfolio website
- Hosted on AWS S3
- Fast and secure delivery using CloudFront CDN
- Public access configuration for static hosting

⚙️ Setup Steps

1. Created an S3 bucket and uploaded website files
2. Enabled static website hosting in S3
3. Set bucket permissions for public access
4. Created a CloudFront distribution
5. Connected CloudFront with S3 bucket
6. Accessed the website using CloudFront URL
